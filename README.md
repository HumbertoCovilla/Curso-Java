# Curso-Java



\# 🚀 Curso de Java: De Cero a Ingeniero



Bienvenido al repositorio central de nuestro curso de aprendizaje colaborativo. Este proyecto está diseñado bajo la metodología \*\*"Aprender Enseñando"\*\*, donde estructuramos los conocimientos de programación en Java de forma progresiva y profesional.



\---



\## 🛠️ Requisitos del Entorno

Para que todo funcione en tu equipo, asegúrate de tener:

\* \*\*JDK:\*\* Versión 17 (LTS).

\* \*\*IDE:\*\* IntelliJ IDEA Community Edition o VS Code.

\* \*\*Git:\*\* Instalado y configurado con tu cuenta de GitHub.



\---



\## 🗺️ Mapa de Aprendizaje



\### 🟢 Nivel 1: Principiante (Fundamentos)

\* Lógica de programación y pseudocódigo.

\* Variables, constantes y tipos de datos primitivos.

\* Operadores aritméticos, lógicos y relacionales.

\* Estructuras de control (`if-else`, `switch`, `for`, `while`).



\### 🟡 Nivel 2: Intermedio (Programación Orientada a Objetos)

\* Conceptos de Clases y Objetos.

\* Los 4 pilares de la POO: Abstracción, Encapsulamiento, Herencia y Polimorfismo.

\* Manejo de Arreglos (`Arrays`) y Listas (`ArrayList`).

\* Gestión de Excepciones (`try-catch`).



\### 🔴 Nivel 3: Avanzado (Ingeniería y Persistencia)

\* Conexión a Bases de Datos (JDBC / MySQL).

\* Patrones de diseño básicos.

\* Principios SOLID y Clean Code.

\* Desarrollo de un sistema CRUD completo con persistencia real.



\---



\## 👨‍💻 Flujo de Trabajo (Scrum Master Rules)

Para mantener el orden como ingenieros, seguiremos estas reglas:

1\. \*\*Issues:\*\* Cada tema nuevo debe tener una "Issue" en el tablero.

2\. \*\*Branches:\*\* No trabajamos en `main`. Creamos una rama `feature/nombre-tema`.

3\. \*\*Pull Requests:\*\* Antes de unir código, se debe crear un PR para revisión.

4\. \*\*Story Points:\*\* Evaluamos la dificultad de 1 a 8 puntos.
